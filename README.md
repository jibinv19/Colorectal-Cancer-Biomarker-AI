# AI-based Biomarker Identification in Colorectal Cancer

## Overview
This project uses single-cell RNA sequencing data to identify biomarkers using:
- Seurat (R)
- Autoencoder + Machine Learning (Python)

## Structure
- R_analysis → Seurat pipeline
- python_analysis → Deep learning + ML
- results → figures and outputs

## Tools Used
- R (Seurat)
- Python (scikit-learn, TensorFlow)
- Google Colab

## Dataset
10x Genomics – Human Colorectal Cancer

## Results
- Identified biomarkers: CEACAM6, COL1A1, IGKC
- Accuracy: ~80–83%
